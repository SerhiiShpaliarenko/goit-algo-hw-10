Висновки до Завдання 2 (Метод Монте-Карло для інтеграла)
Опис завдання
Метою було обчислити визначений інтеграл функції `f(x) = x^2` на відрізку `[0, 2]` за допомогою методу Монте-Карло та порівняти отриманий результат з точним аналітичним значенням (отриманим через функцію `quad` бібліотеки SciPy).

Результати розрахунків
**Інтеграл (SciPy / Аналітично):** 2.666667
**Інтеграл (Монте-Карло, 15 000 точок):** ~2.66 (значення варіюється при кожному запуску)

Порівняльний аналіз та висновки
За результатами тестування можна зробити наступні висновки:
1.  **Правильність розрахунків:** Метод Монте-Карло правильно наближає значення інтеграла. Отриманий результат має незначне відхилення від еталонного значення (зазвичай у межах 0.01 - 0.05 при N=15000).
2.  **Точність методу:** Метод є імовірнісним, тому точність напряму залежить від кількості випадкових точок ($N$). Зі збільшенням $N$ похибка зменшується, а результат наближається до аналітичного, що підтверджує Закон великих чисел.
3.  **Ефективність:** Для простих одновимірних функцій (як $x^2$) аналітичні методи (або `quad`) є точнішими та швидшими. Однак метод Монте-Карло продемонстрував свою працездатність і є незамінним інструментом для наближеного обчислення багатовимірних інтегралів або площ фігур складної форми, де аналітичні методи не працюють.
